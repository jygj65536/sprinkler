# 🌱 식물 물주기 알림 앱

> 토스 앱인토스 입점을 위한 미니앱 — 식집사들이 물주기 타이밍을 놓치지 않도록 돕는 서비스

---

## Background

토스 앱인토스에 입점할 미니앱으로, 아래 기준을 충족하는 방향으로 개발한다.

- 광고 위주의 저품질 앱이 아닌, 실제 일상에서 쓸 수 있는 퀄리티
- 불필요한 리소스 투입 없이 핵심 기능에 집중

---

## Overview

**Objective**

식집사들이 식물을 키울 때 물주기 타이밍을 잊지 않게 해, 꼭 참고할 수 있는 앱이 된다.

### Problem

- 식물을 키울 때 언제 물을 줘야 하는지 판단하기 어렵다
- 과거에 언제 물을 줬는지 기억하기 어렵다

### Opportunity

- 물을 줬는지 꾸준히 기록하게 한다
- 과거 물주기 타이밍을 기반으로, 줘야 하는 시점에 알림을 보내 잊지 않게 한다
- → 유저의 식물을 살리면 앱을 쓸모있다고 여기고, 이탈하지 않고 계속 사용하게 된다

---

## 화면 구조

크게 네 화면으로 구성된다.

| 화면 | 역할 |
|------|------|
| **Home** (main) | 내 식물 목록 확인 (건강상태 직관적 표시), 새로운 식물 추가 진입 |
| **달력** (main) | 식물별 과거 물주기 내역 및 미래 물주기 일정 조회 |
| **내 식물 상세** (sub) | 식물 메타정보, 나와의 유대 정보, 물주기 데이터 조회 |
| **식물 추가** (sub) | 식물 검색 후 "내 식물"로 등록 |

---

## Features

### 1. 식물 검색 및 추가

- 식물 이름을 검색하면 기본 정보, 사진, 이상적인 물주기 기간을 DB에서 자동으로 불러온다
- 검색 후 "내 식물"로 추가하면 내 식물 목록에서 조회 및 관리할 수 있다

### 2. Home — 내 식물 목록

- 등록한 식물 목록을 한눈에 확인한다
- 물을 많이 줬는지, 덜 줬는지에 따라 식물의 건강상태를 직관적으로 표시한다

### 3. 달력 — 물주기 내역 & 일정

- 달력에 노출할 식물 수를 유저가 직접 선택해 조절할 수 있다
- **과거**: 식물당 물주기 내역을 건강 여부에 따라 색깔로 직관적으로 표시한다
- **미래**: 물주기 가능한 날짜를 range로 표기한다 (생리주기 방식, 시각적 복잡성 고려)

### 4. 내 식물 상세

- **식물 메타정보**: 잘 자라는 환경, 이상적인 물주기 기간
- **나와의 유대 정보**: 최초 등록일, 함께한 기간
- **물주기 데이터**: 마지막으로 물 준 경과일, 과거 물주기 이력, 필요한 물주기 양

### 5. 물주기 알림

마지막으로 물 준 날짜 + 이상적인 물주기 기간을 기준으로 알림을 보낸다. 알림 방식은 세 가지다.

1. 앱 접속 시 메인 화면 배너
2. 푸시 알림
3. 위젯 화면 변경
