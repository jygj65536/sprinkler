# 농촌진흥청 농사로 – 실내정원용 식물 Open API 매뉴얼

베이스 URL: `http://api.nongsaro.go.kr/service/garden`

---

## 1. 개요

농촌진흥청 농사로 OpenAPI는 다양한 농업기술정보를 유용하게 활용할 수 있도록 공개한 개발자 인터페이스입니다.

## 2. 이용안내

농촌진흥청 농사로 OpenAPI를 이용하기 위해서는 로그인 후 별도 신청이 필요합니다.  
농사로의 [Open API 이용안내](https://www.nongsaro.go.kr) 페이지에서 API 신청 및 매뉴얼·샘플 파일을 다운로드 받을 수 있습니다.

## 3. 제공 API 서비스

### 3.1 제공 방식

- **REST**: XML 포맷
- **AJAX**: 동적 JavaScript 로딩

### 3.2 텍스트 인코딩

모든 텍스트 데이터는 **UTF-8** 로 제공합니다.

### 3.3 REST XML 포맷 구조

**목록 형식**

```xml
<response>
  <header>
    <resultCode>    <!-- 결과 코드 -->
    <resultMsg>     <!-- 결과 메시지 -->
    <requestParameter>  <!-- 요청 파라미터 -->
  </header>
  <body>
    <items>
      <item> ... </item>   <!-- 서비스 항목 -->
    </items>
    <numOfRows>    <!-- 페이지 당 건수 (기본값: 10) -->
    <pageNo>       <!-- 페이지 번호 (기본값: 1) -->
    <totalCount>   <!-- 총 건수 -->
  </body>
</response>
```

**단건 형식**

```xml
<response>
  <header>
    <resultCode>
    <resultMsg>
    <requestParameter>
  </header>
  <body>
    <item> ... </item>
  </body>
</response>
```
